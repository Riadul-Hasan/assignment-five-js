Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the Document Object Model (DOM). They are: 1. Using query selector. 2.Get element by their id. 3. Get element by their class name.
4. HTML tag name and so on.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Inner HTML: It includes all the set of HTML tag from a body part.
Inner Text: It access the inner text from any HTML element. Only access the text here.
textContent: It contains all the text content without HTMl tags and it also access hidden elements.

Question-3: What is event delegation in the DOM?

Event Delegation: When we need to intaract somewhere using Java Script or when we need to do command for the specific portion or element then we use event delegation.
Event delegation gives us the huge object property for our specific tasks.

Question-4: What is event bubbling in the DOM?

Event bubbling means when we generate any event to a specific element then event works there at first. Then when event finished working then the event is naturally bubbled to its parent element one by one untill the root of the html.

Question-5: How do you create, add, and remove elements using JavaScript?

To create element: document.createElement("div")
To add element: We can use appendChild() and prependChild() for adding specific element.
Remove element: We can remove element by div.remove() or we can use removeChild() too. ('div' is one of the example)
